# Integrating-R-Programming-in-SAS-
Integrating R Programming in SAS®

This session will demonstrate the many ways users can integrate R programming in SAS®  . Whether you want to include R code directly into your SAS®  program, switch back and forth between languages, or access SAS®  Viya®  within your R program, this is the workshop for you. For SAS 9 users, we will discuss how to use PROC IML to integrate R code within a SAS program and the various functions available to read SAS files into R. For SAS Viya users, we will discuss the new PROC R procedure, SAS®  Viya®  Workbench, and the SWAT R package. 

By integrating R and SAS together, you can access the strengths of both languages within the same program and interface.
